# schema

Pydantic v1 data models for the dataflow configuration JSON schema.

Defines `FieldDefinition`, `CobrixOptions`, `InputConfig`, `OutputConfig`, `TransformationStep`, `TransformationConfig`, and the top-level `DataFlowConfig` model. All models use Pydantic v1 API (`.dict()`, `@validator`).
