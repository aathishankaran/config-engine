# keywords

COBOL verb and JCL keyword reference constants.

Contains `COBOL_VERB_TO_PYSPARK` (mapping COBOL verbs like MOVE, ADD, COMPUTE to PySpark equivalents) and `JCL_DD_KEYWORDS` (mapping JCL DD parameter keywords to functional categories).
